# Personal NixOS Configurations

## Usage

Flake support is required.
To build the system config, use the `.#nixosConfigurations.*` flake outputs.
For example,

```sh
nixos-rebuild switch --flake .#x13 --use-remote-sudo
```


## Project Structure

- `flake.nix`: Configs for different machines.
- `default-modules/`: Modules included for all machines.
- `gui-modules/`: Extra modules for GUI-enabled machines.
- `opt-modules/`: Optional modules that requires opting in from `flake.nix`.
- `configs/`: Literal config files.
- `hardware/`: Hardware-specific configs generated by `nixos-generate-config`.
- `services/`: Custom services.
- `packages/`: Custom derivations, available as `.#packages.*`.
